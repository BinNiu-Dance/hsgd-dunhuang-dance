# Code Release Notes

This repository currently releases the model implementation and a clean deep-learning project skeleton.

Included:

- Core HSGD model in `src/hsgd/model.py`.
- Dataset metadata template in `src/hsgd/bvh_utils.py.example`.
- README figures in `assets/images/`.
- Minimal configuration and smoke-test scaffolding.

Not included:

- Raw BVH files or derived private preprocessing outputs.
- Full training and evaluation scripts.
- Trained checkpoints.
- Manuscript source files.
